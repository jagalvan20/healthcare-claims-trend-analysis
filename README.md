# Healthcare Claims Trend Analysis

## Overview
This project analyzes accounts receivable aging and claim denial categories to identify operational bottlenecks in healthcare revenue cycle workflows.

## Objectives
- Evaluate AR aging distribution (0–30, 31–60, 61–90, 120+ days)
- Categorize denial reasons
- Identify recurring reimbursement patterns
- Support process improvement insights

## Tools Used
- Excel (Data structuring and trend evaluation)
- SQL (Foundational query practice)
- Power BI (Academic dashboard development)

## Key Insights
- Identified denial categories contributing to delayed payments.
- Analyzed aging distribution to determine bottlenecks.
- Highlighted areas for improved documentation and policy alignment.
